# Cafe Tracker

A Python application that tracks spending at my university's dining facilities and local campus merchants.

The application imports bank transaction data, stores it in SQLite, and generates spending analytics.

## Features

- Import transactions from CSV
- Store transactions in SQLite
- Prevent duplicate imports
- Calculate:
  - Total spending
  - Average transaction amount
  - Monthly spending
- Merchant filtering

## Technologies

- Python
- Pandas
- SQLite
- VS Code
